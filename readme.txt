=== Description ===

A wordpress plugin for Novinhub users that helps them publish their wordpress posts to their Social Medias.
They can publish image, video and captions.
Also they can schedule publishing.

Note: You must be a Novinhub user to use this plugin. because you need your Novinhub profile access token to use it.

# Additional Important Notes

In order to take advantage of the plugin you should figure out these notes:

* Publish Later
You can set an optional date to publish your post. to do, you should check the 'Publish Later' and
pick a date you want.

* Added Media
If you want to add media to your post, you can do this through Featured Image section or
image or video blocks easily.
In Woocommerce product page, you can use only "product image" as your product image.

=== Installation ===

To install the plugin in order to enjoy using it you can easily follow steps below:
1. Download the plugin Zip file.
2. Upload The Zip file to your wordpress plugins ('/wp-content/plugins/') directory.
3. Activate it.

== Frequently Asked Questions ==

= How do i start Novinhub plugin? =

What a good question. to do this, you should follow steps below:

1. In Plugins menu on your wordpress admin panel, Activate Novinhub plugin.
2. After Activation, the Novinhub menu item will appear. click on it to open Novinhub settings page.
3. In Novinhub settings page, enter your Novinhub access token which provided by your Novinhub dashboard.
4. Click on 'Set Token' to get all of your accounts assigned to your token.
5. If anything is OK, your assigned accounts will appear in a table.
6. Everything is OK. Now you can use the plugin.

= How do i publish my wordpress post to my social Medias? =

In admin edit or new post pages, the novinhub section will appear. you can use it and send posts
to novinhub in order to publish them in your social accounts.

= Can i edit my caption separately before sending? =

Yes, of course...
1. If you click on "Create Caption using post text, get Media and copy Tags from sidebar" button, all texts you've added in Blocks will appear in Novinhub 'Caption Draft' section. also if you've add some image or video you can check it and if you've inserted hashtags in tags area in sidebar menu, those will appear in the bottom of this button. you can
modify your caption draft before you send the post to novinhub.
2. This section control number of characters in the caption. this is necessary if your post destination
has a limit on number of characters.
3. Be careful, All of texts in caption draft will be send to your accounts.

= How do i add media? =

To add media to your post, you can do this through image or video blocks easily.
But you should be alert:
1. If more than one image has been attached, the first one will be uploaded.(Highest Priority: Featured Image)
2. If more than one video has been attached, the first one will be uploaded.
3. If image and video have been attached simultaneously, only the image will be uploaded.

= How do i add some hashtags to send? =

If you want to Add some Hashtags into your caption, you can insert your hashtags into Tags section. (Be sure your hashtags added correctly to wordpress.)

= Can i send my WooCommerce products to my social accounts? =

Yes, to do this, you can easily send them from your Woocommerce product page.
You can send your product title, image, price and description to your social accounts.

= Which Wordpress Editors do you support? =

Our plugin supports default editor on latest wordpress versions (Gutenberg) and Classic editor.
